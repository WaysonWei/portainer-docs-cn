# 添加新注册表

从菜单中选择**注册表**，然后点击**添加注册表**。

<figure><img src="../..//assets/2.19-registries-add.gif" alt=""><figcaption></figcaption></figure>

从**注册表提供商**部分选择您要添加的注册表类型。Portainer为多个流行的注册表提供商提供配置支持：

[dockerhub.md](dockerhub.md)

[ecr.md](ecr.md)

[quay.md](quay.md)

[proget.md](proget.md)

[azure.md](azure.md)

[gitlab.md](gitlab.md)

[ghcr.md](ghcr.md)

您还可以添加自己的自定义注册表：

[custom.md](custom.md)
