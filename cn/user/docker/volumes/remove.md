# 删除数据卷

当您删除数据卷时，数据卷中的所有内容都将被清除。

从菜单中选择 **Volumes**，勾选要删除的数据卷旁边的复选框，然后点击 **Remove**。

您无法删除已附加到容器的数据卷。如果要执行此操作，请先删除容器，然后再删除数据卷。

当确认消息出现时，点击 **Remove**。
